# 투 포인터(Two Pointer) 알고리즘

### 투 포인터 알고리즘이란?

하나의 1차원 배열에 서로 다른 원소를 가리키는 두개의 포인터를 조작해가며 원하는 결과를 나타내는 알고리즘

예제 문제인 백준 20366번을 확인해보자
	-> https://www.acmicpc.net/problem/20366

정리해보자면 N개의 눈덩이 중, 4가지 눈덩이를 선택하여 2개의 눈사람을 만들 때 두 눈사람의 키 차이를 ㄱ
